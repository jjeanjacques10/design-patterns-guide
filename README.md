# Design Patterns Guide

Guia sobre Design Patterns.

### O que é?

Solução comum para um problema recorrente ao utilizar o paradigma da orientação a objectos.

- Strategy
  - Evita o uso excessivo de condicionais
  - Código mais coeso
- Chain of Responsibility
- Template Method
- State
- Command
- Observer
